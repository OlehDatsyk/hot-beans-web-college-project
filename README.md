## 📖 About the Project

**Hot Beans Web** is a responsive multi-page recruitment website developed as my final assignment for college.

The purpose of the website is to simulate a professional company website where visitors can:

- Learn about the company
- Meet the team
- Explore available job roles
- View web development courses
- Apply for jobs online

The project demonstrates the use of modern front-end web development techniques using only HTML, CSS and JavaScript.
